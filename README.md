# TechnoTP Website

Website for TechnoTP.

## Project Structure

```
├── src/
│   ├── css/          # Stylesheets
│   ├── js/           # JavaScript files
│   ├── images/       # Image assets
│   └── fonts/        # Custom fonts
├── public/           # Static files served as-is
├── assets/           # Raw/unprocessed assets
└── index.html        # Main entry point
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/termitemorin/TechnoTP_website.git
   ```
2. Open `index.html` in your browser.
